# EarlyRise — Marketing Website

Landing page for **EarlyRise**, an iOS app that makes you earn your scroll time by completing real-world tasks before unlocking social media.

🍎 [Download on the App Store](https://apps.apple.com/us/app/earlyrise/id6760259048)

---

## About the site

A single-page marketing website covering:

- How EarlyRise works (4-step flow)
- Why it's different from other screen time apps
- Pricing ($2.99/month or $20/year)
- App Store download link

## Tech

- Plain HTML, CSS, and vanilla JavaScript — no frameworks, no build step
- Fonts: Cormorant Garamond + DM Sans via Google Fonts
- Hosted on Netlify with auto-deploy on push to `main`

## Deployment

Any push to `main` automatically deploys to Netlify.

To update the site:
1. Edit `index.html`
2. Commit and push to `main`
3. Netlify handles the rest
